## Human Input

The human reviewed the live UI and identified that the dashboard initially made only the latest profile feel accessible.

## UI Decisions

The dashboard was changed to show a clickable profile directory. Selecting a profile updates:

- Dashboard summary
- Graph
- Timeline
- Repository note context
- Search context

Search evidence was also changed to compact expandable source cards instead of dumping large blocks of text.

## AI Assistance

The AI implemented profile cards, selected-profile loading, source evidence expansion, note removal buttons, and improved search result layout.

## Human Review

The human tested profile switching and noted when Cloud Engineer and SOC Analyst contexts became mixed. The AI fixed scoped search and stale-profile recovery.
