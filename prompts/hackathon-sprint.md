# LegacyOS Hackathon Sprint

Ignore all future enterprise phases.

Objective:

Deliver a fully functional demo within one week.

Success is measured by:

1. Working product
2. Clear problem solved
3. Impressive demonstration
4. Strong AI functionality
5. Strong visual presentation

Not by completeness.

---

Build LegacyOS Lite.

Core Features Only:

1. AI Interview Engine
2. Knowledge Repository
3. Knowledge Timeline
4. Knowledge Graph Visualization
5. Knowledge Risk Score
6. AI Knowledge Search
